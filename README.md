## Project Background

This project is based on the software framework provided in the following repository:

- https://github.com/eljuguetero/T113_SAXO_OS

The repository provides the necessary files to deploy a **Debian-based operating system** on a development board built around the **Allwinner T113 processor**. During the development of this project, several components of the embedded system were modified to adapt it to our hardware and project requirements.

## Hardware Design

The **SoC board PCB design** used in this project is attributed to **Carlos Ivan Camargo Bareño**, who provided the original hardware design used as the base for this development board.

The **saxophone PCB** used in this project is based on the following open hardware repository:

- https://github.com/cardonabits/haxo-hw

This project includes an additional PCB designed to support new requirements and extended functionalities.
